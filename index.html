HTML Form Reference

Form Tags

Tag Name Descrption
<form> Form Is used to create an HTML form for user input
<input> Input Accepts data from the user.
<label> Label Defines a label for several form elements.
<select> Select Defines a drop-down list.
<textarea> Text Area Defines a multi-line input field (a text area).
<button> Button Defines a clickable button
<fieldset> Fieldset Is used to group related data in a form
<legend> Legend Defines a caption for the <fieldset> element
<datalist> Data List Specifies a list of pre-defined options for an <input> element.
<output> Output Represents the result of a calculation (like one performed by a script).
<option> Option Is used to define an item contained in a <select>, an <optgroup>, or a <datalist> element.
<optgroup> Option Group Creates a grouping of options within a <select> element.

Form Attributes

Attribute Name Description
action Action Defines the action to be performed when the form is submitted.
target Target Specifies where to display the response that is received after submitting the form.
method Method Specifies the HTTP method to be used when submitting the form data.
autocomplete Autocomplete Specifies whether a form should have autocomplete on or off.
novalidate No Validate specifies that the form-data (input) should not be validated when submitted.
